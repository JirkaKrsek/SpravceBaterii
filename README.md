# Správce baterií

Tato webová aplikace slouží ke správě spotřebních baterií (nenabíjecích i nabíjecích).

Aplikace umožňuje uživatelům evidovat umístění, zařízení a baterie.

Poskytuje přehled o stavu baterií.


## Technologie
- **C#**
- **.NET Blazor Server**
- **Microsoft SQL Server (MSSQL)**
- **Entity Framework Core**
- **CSS**

## Nasazení
Aplikace bude po dokončení nasazena na webový hosting pro veřejné použití.

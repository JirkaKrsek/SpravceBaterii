# Správce baterií

Webová aplikace pro správu spotřebních baterií - jednorázových i nabíjecích.
Umožňuje evidovat umístění, zařízení a jednotlivé baterie a poskytuje přehled o jejich stavu.

## Použité technologie
- **C#**
- **.NET 10**
- **Blazor Server**
- **Microsoft SQL Server (MSSQL)**
- **Entity Framework Core**
- **ASP.NET Core Identity**
- **CSS**

## Dostupnost
Aplikace je nasazena na pro veřejné použití. Aktuální odkaz je uveden v popisu repozitáře.

## Lokální spuštění projektu
Podrobný postup pro lokální spuštění (instalace, konfigurace a vytvoření databáze) je uveden v sekci **Issues -> Informace**.

## Databázový model
Model databáze je dostupný v sekci **Issues -> Model databáze**.

## Dotazy, návrhy a hlášení chyb
Pro dotazy, návrhy a hlášení chyb vytvářejte nová **Issues** v tomto repozitáři.
